FAQ

What is ChatGPT ?

The ChatGPT model is a large language model trained by OpenAI that is capable of generating human-like text. By providing it with a prompt, it can generate responses that continue the conversation or expand on the given prompt.

What's Awesome ChatGPT Prompts ?

Awesome ChatGPT Prompts is a collection of prompt examples to be used with the ChatGPT model.

How to use the extension ?

Install it at first. Then go to ChatGPT web page. Typing / in the input and you can explore it.
